## English

This Privacy Policy describes how the **SolusVM VPS Dashboard** extension (referred to as "the Extension") handles user information.

### 1. What Data We Collect
The Extension **does not collect, store, or transmit any personal data, telemetry, or browsing history** to any third parties or remote servers. 

### 2. What Data We Store Locally
The Extension stores your API credentials (including API URL, API Key, and API Hash) and configuration preferences locally on your device using Chrome's secure storage API (`chrome.storage.local`). This information is required to communicate with your SolusVM server management panel.

### 3. Network Requests & Third Parties
All network requests generated by the Extension are sent **directly** to the SolusVM API endpoints specified by you in the settings. No data is routed through or cached on any proxy server or third-party service.

### 4. Data Security & Control
Since all data is stored locally on your device, you have complete control over it. You can clear all saved credentials and configurations at any time by:
- Clicking the "Delete" option in the settings page.
- Uninstalling the Extension from your Chrome browser.

### 5. Contact Info
If you have any questions or feedback regarding this Privacy Policy, please open an issue at:
[GitHub Repository](https://github.com/bingege2025/SolusVM-Sidebar-Dashboard)

## 简体中文

本隐私政策描述了 **SolusVM VPS Dashboard** 浏览器扩展（以下简称“本扩展”）如何处理用户信息。

### 1. 收集的数据
本扩展**不收集、不存储、也不向任何第三方或远程服务器传输您的个人数据、遥测数据或浏览历史**。

### 2. 本地存储的数据
本扩展使用 Chrome 的本地安全存储 API (`chrome.storage.local`) 将您的服务器配置信息（包括 API URL、API Key、API Hash 等 API 凭证）以及语言首选项存储在您的本地设备上。这些信息仅用于向您自己的 SolusVM 服务器管理面板发起请求。

### 3. 网络请求与第三方
本扩展产生的所有网络请求都是**直接**发送到您在配置页面中填写的 SolusVM API 接口。任何数据都不会通过第三方代理服务器，也不会在任何第三方服务中缓存。

### 4. 数据安全与用户控制
由于所有配置数据都存储在您的本地设备中，您对此拥有完全控制权。您可以通过以下方式随时清除所有保存的凭证和配置：
- 在扩展程序的选项/设置页面中点击删除服务器。
- 从您的 Chrome 浏览器中卸载本扩展程序。

### 5. 联系我们
如果您对本隐私政策有任何疑问或建议，请在我们的项目主页提交反馈：
[GitHub 项目主页](https://github.com/bingege2025/SolusVM-Sidebar-Dashboard)
